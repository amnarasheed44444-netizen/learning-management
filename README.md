# Learning Management System (LMS) - C#

## 📌 Project Description
This is a simple Console-Based Learning Management System built using C#.  
It allows users to register, login, and access role-based menus (Admin, Teacher, Student).

---

## 🚀 Features
- User Registration
- User Login Authentication
- Role-based Access S
- ystem
- Console-based interactive menu
- Continuous program loop until exit

---

## 🧠 How It Works

- Program starts from `Program.cs`
- Displays a menu:
  - 1. Register
  - 2. Login
  - 3. Exit
- Registration is handled by `AuthService.Register()`
- Login is handled by `AuthService.Login()`
- After successful login, `Menu.Show(user)` is displayed based on user role
